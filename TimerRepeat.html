TimerRepeat

  • setInterval 다루기 쉽도록 객체화

construct

  • var tr = new TimerRepeat();

methods

  • .start([callback,[interval]])
    callback 함수를 interval 마다 실행한다.
    각 값이 null 이면 lastCallback, lastTimeout 을 사용한다.
  • .changeInterval(interval)
    interval 를 바꿔서 다시 실행
  • .run(callback,interval)
    callback 함수를 interval 마다 실행한다.
    아규멘트 값이 없으면 에러!
  • .stop()
    setInterval 를 중지한다.

properties

  • .lastCallback = null or Function
    마지막 동작 함수
  • .lastInterval = null or int
    마지막 동작 interval
  • .tm = null or int
    setTimeout의 timeoutID
